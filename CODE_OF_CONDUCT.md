# Code of Conduct

## 我们的承诺

参与者应以尊重、专业和建设性的方式协作，不因年龄、身体状况、种族、性别认同与表达、经验水平、国籍、外貌、宗教、性取向或技术背景而骚扰或贬低他人。

## 期望行为

- 就事实、代码和风险进行讨论，不进行人身攻击；
- 尊重不同经验水平，提供可复现的信息和耐心的技术说明；
- 对可能造成数据丢失或凭据泄露的建议保持谨慎；
- 在发现错误时及时更正，并保护报告者与用户的敏感信息。

## 不可接受行为

- 骚扰、威胁、歧视、羞辱或持续打扰；
- 未经许可公开他人的私人信息；
- 诱导用户提交密码、完整序列号、私钥或其他敏感数据；
- 明知可能破坏数据仍隐瞒风险或鼓励未经备份的测试。

## 执行

维护者可以编辑或删除不符合本准则的评论、Issue、提交和其他贡献，并可临时或永久限制参与。

行为问题优先通过仓库的 GitHub 私密漏洞报告渠道联系维护者。如果该渠道尚未启用，请仅创建不含敏感细节的公开 Issue，请求维护者提供私密联系途径。
