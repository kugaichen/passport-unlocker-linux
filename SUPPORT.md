# Support

## 使用问题

请先阅读 [README.md](README.md)、[docs/COMPATIBILITY.md](docs/COMPATIBILITY.md) 和 [docs/VALIDATION.md](docs/VALIDATION.md)。当前项目处于 pre-alpha，新的 Linux 实现尚未在真实 WD 硬盘上完成解锁验证。

普通问题可以创建 GitHub Issue。报告中请提供：

- 应用版本；
- Linux 发行版和桌面环境；
- 硬盘型号与 VID:PID；
- 仅序列号后四位；
- 可复现步骤和已清理日志。

## 不要公开的信息

不要提交硬盘密码、密码 hash、完整序列号、原始解锁 payload、私钥、Token 或包含个人路径/凭据的完整日志。

疑似安全漏洞请遵循 [SECURITY.md](SECURITY.md)。数据恢复、密码破解和绕过认证不属于本项目支持范围。
